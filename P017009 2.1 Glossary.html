Notre système MPMS est développé selon le principe OSOT – One Source Of Truth : l’objectif est qu’il n’y ait aucune redondance négative, et que toutes les informations fassent référence à une seule source cohérente.

Sur la capture d’écran ci-jointe, vous retrouvez déjà plusieurs repères utiles :

  • 1 : l’ensemble de votre fichier Excel est intégré dans le système : vos numéros HNS ainsi que les descriptions sont repris dans les colonnes HNS_CLT et LABEL_CLT.
  • 2 : choix du layout (mise en forme de la vue). Vous pouvez sélectionner d’autres layouts via cette liste déroulante.
  • 3 : la barre de recherche. Tapez un mot (ou une partie de mot) et les lignes correspondantes s’affichent immédiatement (si vous tapez un espace à la fin, la recherche devient « contient »).
  • 4 : le bouton Save pour enregistrer vos modifications.
  • 5 : l’icône « main avec stylo » indique la présence d’une note.
  • 6 : l’icône « personnes » indique une réunion.
  • 7 : l’icône de document indique qu’un fichier est lié à la ligne.
  • 8 : l’icône de lien pointe vers un document ou une autre ressource. Lorsque le lien mène vers un espace interne à MEAM, il est possible que vous n’y ayez pas accès. Pour la plupart des cas, il s’agit de fichiers PDF lorsque le document est déjà disponible.

Quelques astuces de navigation :

  • Clic droit sur une ligne pour l’étendre ou la réduire (expand / collapse).
  • Clic droit sur les en-têtes de colonnes pour afficher ou masquer d’autres champs.
